Az alkalmazás futtatása:

1. Nyisd meg az API mappát.

2. Futtasd a kódot a terminálban a következő paranccsal:
py -m uvicorn main:app --reload --port 9000

(Lehet használni a pdf feladatleírásban szereplő virtuális környezetet is.)

3. A böngésző címsorába írd be a következő URL-t:
http://127.0.0.1:9000/docs

4. A kapott interaktív Swagger UI felületen tudod tesztelni az egyes API végpontokat. Teszteld a Root végpontot.

5. A feladat megoldását a "Python Beadandó feladat 1.pdf" fájlban leírtaknak megfelelően készítsd el.
